# Spring Security

이 서비스는 패스트캠퍼스의 한 번에 끝내는 Spring 완.전.판 초격차 패키지 Online 강의의 실습용 프로젝트로 사용되었습니다.

## 개발 환경
* Intellij IDEA
* Java 16
* Gradle 7.1.1
* Spring Boot 2.5.3

## 기술 세부 스택
* Spring Boot
* Spring Boot Actuator
* Spring Data JPA
* Rest Repositories HAL Explorer
* Validation
* Spring Web
* Thymeleaf
* Spring Security
* H2 Database
* MySQL Driver
* Lombok
* Spring Configuration Processor
